00228  \section BUILD_WINDOWS Build instructions for x86 Win32 Platform (HOST Emulation Mode)
00229 
00230  \par Requirements
00231   - GCC and Windows
00232     - Currently the build is tested with "mingw" that is installed as part of CodeBlocks IDE
00233     - Install "CodeBlocks IDE" 16.01 or higher [http://www.codeblocks.org/]
00234     - GCC compiler with 32b mode support is required for using TIOVX
00235       - GCC compiler included as part of Codeblocks is 32b compiler
00236     - Make sure xdctools_x_xx_xx_xx_core/bin/gmake available as part
00237       of XDC package (included in Vision SDK) is visible in the system path.
00238 
00239  \par Setup
00240   - Edit $TIOVX_PATH/vsdk_tools_path.mak
00241     - Modify "GCC_WINDOWS_ROOT := C:/CodeBlocks/MinGW" to match your environment if needed
00242     - The following paths are the only paths that must be set correctly for host emulation mode. (If paths have been already set for build for Vision SDK Platform, they do not need to be changed.)
00243       - VXLIB_PATH
00244       - DMAUTILS_PATH (must only be set if BUILD_BAM option is enabled)
00245       - ALGFRAMEWORK_PATH (must only be set if BUILD_BAM option is enabled)
00246       - EVE_SW_PATH (must only be set if BUILD_IVISION_KERNELS is enabled)
00247   - Edit $TIOVX_PATH/Makefile
00248     - set BUILD_EMULATION_MODE=yes and BUILD_TARGET_MODE=no
00249 
00250  \par Build
00251   - Open a windows command prompt at $TIOVX_PATH
00252   - Type "gmake" to build TIOVX and associated libraries
00253     - This builds the TIOVX libaries, conformance test executable and tutorial executable
00254   - Conformance test, tutorial executable is output at
00255     - "$TIOVX_PATH\out\PC\X86\WINDOWS\debug\vx_conformance_tests_exe.exe"
00256     - "$TIOVX_PATH\out\PC\X86\WINDOWS\debug\vx_tutorial_exe.exe"
00257 
00258  \par Running the conformance test and tutorial
00259   - Open a command prompt at the folder "$TIOVX_PATH\out\PC\X86\WINDOWS\$PROFILE\"
00260   - Set environment variable below to specify the path of the test data. This is used by both
00261     the conformance test and tutorial executables
00262     \code
00263     set VX_TEST_DATA_PATH=../../../../../conformance_tests/test_data
00264     \endcode
00265   - Execute "vx_conformance_tests_exe.exe" to run the conformance test
00266   - Execute "vx_tutorial_exe.exe" to run the tutorial
00267   - NOTE: Output .bmp files generated by the tutorial will be present in the folder $VX_TEST_DATA_PATH
