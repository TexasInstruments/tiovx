00126  \section BUILD_WINDOWS Build instructions for x86 Win32 Platform (HOST Emulation Mode)
00127 
00128  \par Requirements
00129   - GCC and Windows
00130     - Currently the build is tested with "mingw" that is installed as part of CodeBlocks IDE
00131     - Install "CodeBlocks IDE" 16.01 or higher [http://www.codeblocks.org/]
00132     - GCC compiler with 32b mode support is required for using TIOVX
00133       - GCC compiler included as part of Codeblocks is 32b compiler
00134     - Make sure xdctools_x_xx_xx_xx_core/bin/gmake available as part
00135       of XDC package (included in Vision SDK) is visible in the system path.
00136 
00137  \par Setup
00138   - Edit <TIOVX_PATH>/tools_path.mk
00139     - Modify "GCC_WINDOWS_ROOT := C:/CodeBlocks/MinGW" to match your environment if needed
00140     - Modify below paths to point to your environment
00141       - When TIOVX is installed as part of Vision SDK below paths dont need to be changed.
00142         - EDMA3_LLD_PATH
00143         - DMAUTILS_PATH
00144         - ALGFRAMEWORK_PATH
00145         - VXLIB_PATH
00146   - Edit <TIOVX_PATH>/Makefile
00147     - set BUILD_EMULATION_MODE=yes and BUILD_TARGET_MODE=no
00148 
00149  \par Build
00150   - Open a windows command prompt at <TIOVX_PATH>
00151   - Type "gmake -s -j" to build TIOVX and associated libraries
00152     - This builds the TIOVX libaries, conformance test executable and tutorial executable
00153   - Conformance test executable is output at
00154     - "<TIOVX_PATH>\out\PC\X86\WINDOWS\debug\vx_conformance_tests_exe.exe"
00155   - Tutorial executable is output at
00156     - "<TIOVX_PATH>\out\PC\X86\WINDOWS\debug\vx_tutorial_exe.exe"
00157 
00158  \par Running the conformance test and tutorial
00159   - Open a command prompt at the folder "<TIOVX_PATH>\out\PC\X86\WINDOWS<PROFILE>\"
00160   - Set environment variable below to specify the path of the test data. This is used by both
00161     the conformance test and tutorial executables
00162     - set VX_TEST_DATA_PATH=../../../../../conformance_tests/test_data
00163   - Execute "vx_conformance_tests_exe.exe" to run the conformance test
00164   - Execute "vx_tutorial_exe.exe" to run the tutorial
00165   - NOTE: Output .bmp files generated by the tutorial will be present in SD card in the folder $(VX_TEST_DATA_PATH)
00166  */
