223  \section BUILD_WINDOWS Build instructions for x86 Win32 Platform (HOST Emulation Mode)
224 
225  \par Requirements
226  - GCC and Windows
227  - Currently the build is tested with "mingw" that is installed as part of CodeBlocks IDE
228  - Install "CodeBlocks IDE" 16.01 or higher [http://www.codeblocks.org/]
229  - GCC compiler with 32b mode support is required for using TIOVX
230  - GCC compiler included as part of Codeblocks is 32b compiler
231  - Make sure xdctools_x_xx_xx_xx_core/bin/gmake available as part
232  of XDC package (included in Vision SDK) is visible in the system path.
233 
234  \par Setup
235  - Edit $TIOVX_PATH/vsdk_tools_path.mak
236  - Modify "GCC_WINDOWS_ROOT := C:/CodeBlocks/MinGW" to match your environment if needed
237  - The following paths are the only paths that must be set correctly for host emulation mode. (If paths have been already set for build for Vision SDK Platform, they do not need to be changed.)
238  - VXLIB_PATH
239  - DMAUTILS_PATH (must only be set if BUILD_BAM option is enabled)
240  - ALGFRAMEWORK_PATH (must only be set if BUILD_BAM option is enabled)
241  - EVE_SW_PATH (must only be set if BUILD_IVISION_KERNELS is enabled)
242  - Edit $TIOVX_PATH/Makefile
243  - set BUILD_EMULATION_MODE=yes and BUILD_TARGET_MODE=no
244 
245  \par Build
246  - Open a windows command prompt at $TIOVX_PATH
247  - Type "gmake" to build TIOVX and associated libraries
248  - This builds the TIOVX libraries, conformance test executable and tutorial executable
249  - Conformance test, tutorial executable is output at
250  - "$TIOVX_PATH\out\PC\X86\WINDOWS\debug\vx_conformance_tests_exe.exe"
251  - "$TIOVX_PATH\out\PC\X86\WINDOWS\debug\vx_tutorial_exe.exe"
252 
253  \par Running the conformance test and tutorial
254  - Open a command prompt at the folder "$TIOVX_PATH\out\PC\X86\WINDOWS\$PROFILE\"
255  - Set environment variable below to specify the path of the test data. This is used by both
256  the conformance test and tutorial executables
257  \code
258  set VX_TEST_DATA_PATH=../../../../../conformance_tests/test_data
259  \endcode
260  - Execute "vx_conformance_tests_exe.exe" to run the conformance test
261  - Execute "vx_tutorial_exe.exe" to run the tutorial
262  - NOTE: Output .bmp files generated by the tutorial will be present in the folder $VX_TEST_DATA_PATH
263
