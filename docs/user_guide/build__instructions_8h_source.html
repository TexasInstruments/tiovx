00158  \section BUILD_WINDOWS Build instructions for x86 Win32 Platform (HOST Emulation Mode)
00159 
00160  \par Requirements
00161   - GCC and Windows
00162     - Currently the build is tested with "mingw" that is installed as part of CodeBlocks IDE
00163     - Install "CodeBlocks IDE" 16.01 or higher [http://www.codeblocks.org/]
00164     - GCC compiler with 32b mode support is required for using TIOVX
00165       - GCC compiler included as part of Codeblocks is 32b compiler
00166     - Make sure xdctools_x_xx_xx_xx_core/bin/gmake available as part
00167       of XDC package (included in Vision SDK) is visible in the system path.
00168 
00169  \par Setup
00170   - Edit $TIOVX_PATH/tools_path.mk
00171     - Modify "GCC_WINDOWS_ROOT := C:/CodeBlocks/MinGW" to match your environment if needed
00172     - Modify below paths to point to your environment
00173       - When TIOVX is installed as part of Vision SDK below paths dont need to be changed.
00174         - EDMA3_LLD_PATH
00175         - DMAUTILS_PATH
00176         - ALGFRAMEWORK_PATH
00177         - VXLIB_PATH
00178   - Edit $TIOVX_PATH/Makefile
00179     - set BUILD_EMULATION_MODE=yes and BUILD_TARGET_MODE=no
00180 
00181  \par Build
00182   - Open a windows command prompt at $TIOVX_PATH
00183   - Type "gmake" to build TIOVX and associated libraries
00184     - This builds the TIOVX libaries, conformance test executable and tutorial executable
00185   - Conformance test, tutorial executable is output at
00186     - "$TIOVX_PATH\bin\PC\X86\WINDOWS\debug\vx_conformance_tests_exe.exe"
00187     - "$TIOVX_PATH\bin\PC\X86\WINDOWS\debug\vx_tutorial_exe.exe"
00188 
00189  \par Running the conformance test and tutorial
00190   - Open a command prompt at the folder "$TIOVX_PATH\bin\PC\X86\WINDOWS\$PROFILE\"
00191   - Set environment variable below to specify the path of the test data. This is used by both
00192     the conformance test and tutorial executables
00193     \code
00194     set VX_TEST_DATA_PATH=../../../../../conformance_tests/test_data
00195     \endcode
00196   - Execute "vx_conformance_tests_exe.exe" to run the conformance test
00197   - Execute "vx_tutorial_exe.exe" to run the tutorial
00198   - NOTE: Output .bmp files generated by the tutorial will be present in the folder $VX_TEST_DATA_PATH
