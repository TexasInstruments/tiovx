272  \section BUILD_WINDOWS Build instructions for x86 Win32 Platform (HOST Emulation Mode)
273 
274  \par Requirements
275  - GCC and Windows
276  - Currently the build is tested with "mingw" that is installed as part of CodeBlocks IDE
277  - Install "CodeBlocks IDE" 16.01 or higher [http://www.codeblocks.org/]
278  - GCC compiler with 32b mode support is required for using TIOVX
279  - GCC compiler included as part of Codeblocks is 32b compiler
280  - Make sure xdctools_x_xx_xx_xx_core/bin/gmake available as part
281  of XDC package (included in Vision SDK) is visible in the system path.
282 
283  \par Setup
284  - Edit $TIOVX_PATH/vsdk_tools_path.mak
285  - Modify "GCC_WINDOWS_ROOT := C:/CodeBlocks/MinGW" to match your environment if needed
286  - The following paths are the only paths that must be set correctly for host emulation mode. (If paths have been already set for build for Vision SDK Platform, they do not need to be changed.)
287  - VXLIB_PATH
288  - DMAUTILS_PATH (must only be set if BUILD_BAM option is enabled)
289  - ALGFRAMEWORK_PATH (must only be set if BUILD_BAM option is enabled)
290  - EVE_SW_PATH (must only be set if BUILD_IVISION_KERNELS is enabled)
291  - Edit $TIOVX_PATH/Makefile
292  - set BUILD_EMULATION_MODE=yes and BUILD_TARGET_MODE=no
293 
294  \par Build
295  - Open a windows command prompt at $TIOVX_PATH
296  - Type "gmake" to build TIOVX and associated libraries
297  - This builds the TIOVX libraries, conformance test executable and tutorial executable
298  - Conformance test, tutorial executable is output at
299  - "$TIOVX_PATH\out\PC\X86\WINDOWS\debug\vx_conformance_tests_exe.exe"
300  - "$TIOVX_PATH\out\PC\X86\WINDOWS\debug\vx_tutorial_exe.exe"
301 
302  \par Running the conformance test and tutorial
303  - Open a command prompt at the folder "$TIOVX_PATH\out\PC\X86\WINDOWS\$PROFILE\"
304  - Set environment variable below to specify the path of the test data. This is used by both
305  the conformance test and tutorial executables
306  \code
307  set VX_TEST_DATA_PATH=../../../../../conformance_tests/test_data
308  \endcode
309  - Execute "vx_conformance_tests_exe.exe" to run the conformance test
310  - Execute "vx_tutorial_exe.exe" to run the tutorial
311  - NOTE: Output .bmp files generated by the tutorial will be present in the folder $VX_TEST_DATA_PATH
