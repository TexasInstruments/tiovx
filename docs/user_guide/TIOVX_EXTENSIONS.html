TI Extensions to the OpenVX Specification
TI Extensions
  • The TI Extensions to the OpenVX Specification are broken down into the External API and Internal API. The External API is accessible to the user while the Internal API is used only in the framework. The External API is further broken down into the Host API, Target API, and Common API.
