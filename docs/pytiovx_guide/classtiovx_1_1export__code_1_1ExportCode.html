Detailed Description

Export objects from context to C source code.

Example Usage: Export objects from context to C source code
1 from tiovx import *
2 
3 my_context = Context("my_usecase")
4 
5 ...
6 
7 ExportCode(my_context, VISION_APPS_PATH).export()

Output files from the above parameters:

1 <VISION_APPS_PATH>/apps/my_usecase/DEVELOPER_TODO.txt
2 <VISION_APPS_PATH>/apps/my_usecase/concerto_inc.mak
3 <VISION_APPS_PATH>/apps/my_usecase/app_my_usecase/concerto.mak
4 <VISION_APPS_PATH>/apps/my_usecase/app_my_usecase/my_usecase.c
5 <VISION_APPS_PATH>/apps/my_usecase/app_my_usecase/my_usecase.h

Output files from the above parameters in the case that the environment variable CUSTOM_APPLICATION_PATH was given:

1 <CUSTOM_APPLICATION_PATH>/DEVELOPER_TODO.txt
2 <CUSTOM_APPLICATION_PATH>/concerto_inc.mak
3 <CUSTOM_APPLICATION_PATH>/custom_tools_path.mak
4 <CUSTOM_APPLICATION_PATH>/kernels/custom_app_kernel_library_tests.h
5 <CUSTOM_APPLICATION_PATH>/app_my_usecase/concerto.mak
6 <CUSTOM_APPLICATION_PATH>/app_my_usecase/my_usecase.c
7 <CUSTOM_APPLICATION_PATH>/app_my_usecase/my_usecase.h
