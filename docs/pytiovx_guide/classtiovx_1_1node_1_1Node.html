Detailed Description

Node object (OpenVX equivalent = vx_node)

This object is base class for specific nodes which inherit from this class. Some basic checks like parameter matching is done by this class. Inherited node classes can do additional parameter checking.

Node is created by using below syntax.

1 from tiovx import *
2 
3 my_node1 = Node<Name>(<data object param>, <data object param>, ..., name="<string>", target=Target.<Target Name>)
4 
5 # node needs to be added to a graph
6 my_graph.add(my_node1)
7 
8 # a more compact way to the same is
9 my_graph.add( Node<Name>(<data object param>, <data object param>, ..., name="<string>", target=Target.<Target Name>))

Each node can take optional parameter of 'name' and 'target' as input.
'name' is used to given a user readable name to the node which can be seen later in the generated code, image
'target' is used to specify the target on which the node will run when executed. See tiovx.enums.Target for list of supported targets.

Example Usage: Create nodes and add to graph
1 from tiovx import *
2 
3 # create a node Sobel3x3 and add to graph object.
4 # target is specified as DSP1
5 graph.add ( NodeSobel3x3(in_image, grad_x, grad_y, target=Target.DSP1) )
6 
7 # create a node Phase and add to graph object.
8 # target is specified as DSP2
9 graph.add ( NodePhase(grad_x, grad_y, phase, target=Target.DSP2) )
10 
11 # create a custom node, as defined in next example and add to graph
12 graph.add ( NodePhaseRgb(phase, phase_rgb, target=Target.DSP1) )

Users can inherit from Node class to define their own custom nodes.
NOTE: Users need not modify this file to inherit from Node class.
Below code snippet shows one such example

Example Usage: Create custom node class from base class
1 from tiovx import *
2 
3 # inherit custom node class from tiobx.node.Node
4 class NodePhaseRgb (Node) :
5  # implement constructor
6  # first list all input and output parameter data objects
7  # next provide optional parameters of name and target
8  def __init__(self, image_in, image_out, name="default", target=Target.DEFAULT) :
9  # call base class constructor with string of kernel name and list of parameters.
10  # This string of kernel name is later used to create the node
11  Node.__init__(self, "vx_tutorial_graph.phase_rgb", image_in, image_out)
12  # Tell base how many of the parameters are input and how many are output
13  # Also tell the data object type of the parameter
14  # This is used by base class to do type checking
15  self.setParams(1, 1, Type.IMAGE, Type.IMAGE)
16  # Call base class API to set user provided target
17  self.setTarget(target)
18  # Tell base class that this is a user kernel and not a OpenVX specified kernel
19  self.setKernelEnumName("VX_USER_KERNEL")
20 
21  # implement function to do parameter checking
22  # if not implemented then base class will do basic parameter checking like type checking
23  def checkParams(self, *param_type_args) :
24  # first call base class parameter checker
25  Node.checkParams(self, *param_type_args)
26  # Now add additional error conditions over the base class ones
27  assert ( self.ref[0].width == self.ref[1].width ), "Input and Output width MUST match"
28  assert ( self.ref[0].height == self.ref[1].height ), "Input and Output height MUST match"
29  assert ( self.ref[0].df_image == DfImage.U8 ), "Input data format must be U8"
30  assert ( self.ref[1].df_image == DfImage.RGB ), "Output data format must be RGB"
