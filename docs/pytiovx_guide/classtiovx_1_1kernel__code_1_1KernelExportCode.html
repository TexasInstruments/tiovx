Detailed Description

Code object used to generate custom kernel.

Example Usage: Setting up file paths for custom kernel
Note: if leaving the optional field for env_var blank, you must set CUSTOM_KERNEL_PATH
1 from tiovx import *
2 
3 code = KernelExportCode(Module.IMAGING, Core.C66, "CUSTOM_KERNEL_PATH")
4 <set up kernel parameters>
5 code.export(kernel)

Output files from the above parameters:

1 <CUSTOM_KERNEL_PATH>/DEVELOPER_TODO.txt (generated first time only for given parameters)
2 <CUSTOM_KERNEL_PATH>/include/TI/tivx_imaging.h (generated first time only for given parameters)
3 <CUSTOM_KERNEL_PATH>/include/TI/tivx_imaging_kernels.h (generated first time only for given parameters)
4 <CUSTOM_KERNEL_PATH>/include/TI/tivx_imaging_nodes.h (generated first time only for given parameters)

Output folder from the above parameters:

1 <CUSTOM_KERNEL_PATH>/imaging/

This folder contains the following:

1 c66/concerto.mak (generated first time only for given parameters)
2 c66/vx_<kernel_name>_target.c
3 c66/vx_kernels_imaging_target.c (generated first time only for given parameters)
4 c66/bam/vx_bam_<kernel_name>_target.c (if using the C66 DSP)
5 c66/bam/concerto.mak (generated first time only for given parameters, if using the C66 DSP)
6 host/concerto.mak (generated first time only for given parameters)
7 tivx_imaging_node_api.c (generated first time only for given parameters)
8 host/vx_<kernel_name>_host.c
9 host/vx_kernels_imaging_host.c (generated first time only for given parameters)
10 include/tivx_imaging_kernels.h (generated first time only for given parameters)
11 include/tivx_kernel_<kernel_name>.h
12 test/concerto.mak (generated first time only for given parameters)
13 test/test_main.h (generated first time only for given parameters)
